# CocricotLite

鎮守府ModとCocricotModを同時に入れるとブロックの追加量が多すぎてIDが枯渇するという<br>
わけわからなすぎる自体が発生したので、メタデータを使って使用ID量を減らそうと試みてるModです。

Cocricotのコンテンツのみ追加しています。 Cocricotのホームページはこちら: http://cocricot.pics/
